# OIBSIP Task 2 - Personal Portfolio

## Project Title

Personal Portfolio Website

## Objective

The objective of this project is to create a personal portfolio website using HTML and CSS that showcases my profile, technical skills, projects, resume, and contact information.

## Description

This portfolio website is designed to present my academic background, technical expertise, and project experience in a professional manner. The website includes an introduction section, technical skills, project details, resume access, and contact information.

## Features

* Professional portfolio design
* About Me section
* Technical Skills section
* Project showcase
* Resume section
* Contact Information

## Technologies Used

* HTML5
* CSS3

## Projects Included

### Digital Wellbeing Monitor

A Python and Flask based application that tracks PC usage and helps users manage screen time using SQLite for local data storage.

### Face Recognition Based Attendance System

An AI-based attendance management system that automatically recognizes faces and marks attendance.

### Food Waste Management Platform

A MERN stack web application that connects businesses with surplus food to charities and community members to reduce food waste.
